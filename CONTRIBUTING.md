# Contributing to Event-Driven Analytics Platform

Thank you for your interest in contributing! This is a portfolio/educational project, but contributions are welcome.

## How to Contribute

1. **Fork the repository**
2. **Create a feature branch:** `git checkout -b feature/amazing-feature`
3. **Commit your changes:** `git commit -m 'Add amazing feature'`
4. **Push to the branch:** `git push origin feature/amazing-feature`
5. **Open a Pull Request**

## Development Setup

See [QUICKSTART.md](docs/QUICKSTART.md) for local development setup.

## Code Style

- Use TypeScript for all new code
- Follow existing code structure
- Add tests for new features
- Update documentation

## Pull Request Guidelines

- Describe what the PR does
- Include screenshots/videos if UI changes
- Ensure all tests pass
- Update README if needed

## Questions?

Open an issue for discussion before starting major changes.
